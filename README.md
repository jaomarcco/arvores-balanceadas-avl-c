## 💻 Sobre o Projeto
Este projeto contém a implementação em linguagem C de uma **Árvore AVL** (Árvore Binária de Busca Balanceada). O código demonstra na prática:
* Criação da estrutura do Nó (com controle de altura).
* Cálculo do Fator de Balanceamento (FB).
* Implementação das Rotações (Esquerda e Direita) para correção de desbalanceamento.
* Teste de mesa inserindo elementos ordenados (10, 20, 30) para forçar e demonstrar o balanceamento automático da raiz.

## 🚀 Como executar
Para compilar e executar o código em seu terminal, utilize o GCC:

```bash
gcc arvore_avl.c -o arvore_avl
./arvore_avl

```
